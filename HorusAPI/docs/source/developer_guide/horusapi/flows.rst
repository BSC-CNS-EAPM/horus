****************
Predefined Flows
****************

You can add a :bdg-secondary-line:`Flows` folder inside the :bdg-secondary-line:`Plugin` folder.
This folder will be scanned when the plugin is loaded and the flows inside it will be displayed as predefined flows inside the GUI.

.. code-block:: bash

    MyPlugin
    ├── Flows/ # Put here all your predefined flows
    ├── plugin.meta
    └── main.py
