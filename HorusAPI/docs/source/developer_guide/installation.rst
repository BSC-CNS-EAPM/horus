************
Installation
************

In order to develop :bdg-secondary-line:`Plugins` for Horus, you will need to
install the HorusAPI package in your python environment. This package contains the necessary
classes and functions to interact with the Horus app.

Main installation
=================

You will need to download the latest release of |Product| from the
`releases page <https://github.com/NBDsoftware/horus/releases>`_. You can also install
|Product| directly from PyPI using ``pip``:

.. code-block:: bash

    pip install horusapi

Once installed, you can import the |Product| package into your Python
application:

.. code-block:: python

    import HorusAPI

.. note::

    To install from the releases page, perform the following command instead:

    .. code-block:: bash

        pip install <path to horusapi-x.x.x.tar.gz>
