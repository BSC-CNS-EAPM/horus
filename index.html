Horus

The workflow manager built for you

Horus is an innovative multi-platform workflow manager which empowers scientists in areas such as molecular modelling.

HorusAPI

Horus features

Horus comes with several built-in features such as itneractive blocks, Mol* molecular visualizer and powerful SSH support.

Designed to be useful

Integrated with a 2D infinite canvas, Horus excels as a modular workflow manager across different environments. Its autonomous blocks enable seamless linking and unlimited expandability

Build and share your workflows
Workflows can be shared easily with co-workers. Mol* state gets saved along with all the information of your flow
Configure remotes
SSH connections can be configured in order to send calculations to remote machines
Do more with extensions
Additional views can be added to Horus in order to expand its features
Plugins for everyone
Horus comes with a rich Python API which allows for anyone to build their own Blocks and Extensions
